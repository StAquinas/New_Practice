Memory in a Computer

Hard Drive

Hard drives are the a primary storage device for most computers, they are the cheapest form of memory storage to purchase in a computer.

Solid State Drive(SSD)

An SSD offers fast data access and retrieval of memory in a computer. It is more efficient than a solid state drive, but more expensive.

Random Access Memory(RAM)

Random access memory provides temporary storage of information that is going to be used and rewritted over. It will be used for currently open files like an operating system, and any application that might be open
